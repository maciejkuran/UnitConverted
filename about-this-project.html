About This Project

My name is Maciej Kuran-Janowski and I am a polish web designer & developer from Poland. I am proud to build this free unit converter for the community. It was such a great pleasure to get all the things together and push this app to production. The entire process of creation was definitely an amazing experience. Totally worth it.


UnitConverted.com is a free unit converter where you can easily convert all common types of units: length, temperature, area, volume, weight, time and speed. It is a super helpful tool if you expect to have some unit converted fast.


In addition, you can read the definitions of each unit with direct reference to scientific sources.
